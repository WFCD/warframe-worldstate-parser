Class: SentientOutpost

SentientOutpost

Represents a set of sentient outposts that are present Parsed source is combined data from DE's worldstate and semlar.com/anomaly.json

new SentientOutpost()

Properties:
Name Type Description
mission Mission List of current missions
id string Identifier for the mission node with active indicator
active boolean Whether or not the mission is active
activation Date When the mission became or becomes active
expiry Date When the mission became or becomes inactive
previous Object Estimation data for the last mission that was active. Could also be the current.
